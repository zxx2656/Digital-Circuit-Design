# 秒钟（秒表）设计思路

> 基于高云 GW1N-UV9EQ144C6/I5 FPGA 核心板的数字逻辑课程设计

---

## 一、设计需求分析

根据课程设计要求，需要设计一个具有以下功能的秒表：

| 序号 | 功能需求 | 说明 |
|------|----------|------|
| 1 | 计时范围 | 00:00 ~ 59:59（1小时），到达59:59后回零 |
| 2 | 实时显示 | 用共阳极LED数码管实时显示分秒计数值 |
| 3 | 复位功能 | 任意时刻可清零返回 00:00 |
| 4 | 暂停/恢复 | 任意时刻暂停计数并保持显示，可随时恢复 |
| 5 | 预置初值 | 可预置分秒任意时间值（如 15:45） |
| 6 | 状态指示 | 用发光二极管指示工作状态 |

### 硬件条件

| 器件 | 型号/规格 | 数量 | 用途 |
|------|-----------|------|------|
| FPGA核心板 | GW1N-UV9EQ144C6/I5 | 1 | 主控芯片 |
| 4位共阳数码管 | SR410361N | 1 | 显示分秒（mm:ss） |
| 拨动开关 | SK-12D07VG3 | 2~3 | 清零/启动、暂停/恢复、预置 |
| 轻触按键 | 6×6×5mm | 2~4 | 预置值调整 |
| 发光二极管 | 5mm（红/绿） | 3~4 | 秒脉冲、运行状态、暂停指示 |
| 三极管 | S8550 (TO-92) | 4~6 | 数码管位驱动 |
| 排针排母 | 2×20P 2.54mm | 2 | 核心板与底板连接 |
| 电阻 | 470Ω | 8 | LED限流 |
| 电阻 | 10KΩ | 4~6 | 开关上拉 |
| 电阻 | 4.7KΩ | 4~6 | 三极管基极限流 |

## 二、系统总体架构

```
┌─────────────────────────────────────────────────────────┐
│                   秒表系统框图                            │
│                                                         │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐           │
│  │ 按键/开关 │───▶│          │───▶│ LED数码管 │           │
│  │ 输入模块  │    │  FPGA    │    │ 显示模块  │           │
│  └──────────┘    │ GW1N-    │    └──────────┘           │
│                  │ UV9EQ144 │                            │
│  ┌──────────┐    │          │    ┌──────────┐           │
│  │ 晶振时钟  │───▶│          │───▶│ LED指示灯 │           │
│  │ 50MHz    │    │          │    │ 状态显示  │           │
│  └──────────┘    └──────────┘    └──────────┘           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 模块划分

系统共分为 **6 个核心模块**：

1. **时钟分频模块（clk_div）** — 将 50MHz 晶振分频产生 1Hz 秒脉冲
2. **按键消抖模块（key_debounce）** — 消除机械开关抖动
3. **计数器模块（counter_60）** — 实现 0~59 BCD 码计数
4. **控制状态机（fsm_ctrl）** — 管理运行/暂停/清零/预置状态
5. **数码管扫描显示模块（seg_scan）** — 动态扫描驱动共阳极数码管
6. **顶层模块（top_stopwatch）** — 整合所有子模块

## 三、各模块详细设计

### 3.1 时钟分频模块

```verilog
// 50MHz → 1Hz 分频
// 分频系数: 50_000_000 / 2 = 25_000_000
// 计数器范围: 0 ~ 24_999_999
```

- 输入：`clk_50MHz`、`rst_n`
- 输出：`clk_1Hz`（占空比 50% 的秒脉冲信号）
- 关键：需要一个 25 位计数器，计到 24,999,999 时翻转输出

### 3.2 按键消抖模块

- 采样时钟：约 1kHz（从 50MHz 分频得到）
- 消抖时间：约 20ms
- 原理：连续采样 N 次（如 20 次）均为同一电平，才确认按键状态变化
- 输出：稳定的按键/开关信号

### 3.3 60进制 BCD 计数器

```
秒个位：0→1→2→...→9→0（逢10进1）
秒十位：0→1→2→3→4→5→0（逢6进1，到59秒后归零，分钟进1）
分个位：同上
分十位：同上
```

- 4 个独立 BCD 码输出：`sec_l[3:0]`、`sec_h[3:0]`、`min_l[3:0]`、`min_h[3:0]`
- 支持：加1计数、并行预置、同步清零
- 当 `59:59 + 1` 时 → 自动归零到 `00:00`

### 3.4 控制状态机

```
        启动
  ┌──────────────┐
  │              ▼
  │      ┌──────────┐
  │      │  IDLE    │ ◀── 复位
  │      │ (00:00)  │
  │      └────┬─────┘
  │           │ 启动(start=1)
  │      ┌────▼─────┐
  │      │  COUNT   │ ◀─────────┐
  │      │ (正常计时) │           │
  │      └────┬─────┘           │
  │           │ 暂停(pause=1)    │ 恢复(resume=1)
  │      ┌────▼─────┐           │
  │      │  PAUSE   │ ──────────┘
  │      │ (暂停显示) │
  │      └──────────┘
```

**状态跳转条件：**

| 当前状态 | 跳转条件 | 下一状态 | 说明 |
|----------|----------|----------|------|
| IDLE | start = 1 | COUNT | 按启动键开始计时 |
| COUNT | rst = 1 | IDLE | 任意时刻按复位键清零 |
| COUNT | pause = 1 | PAUSE | 暂停计时 |
| PAUSE | resume = 1 | COUNT | 恢复计时 |
| PAUSE | rst = 1 | IDLE | 暂停态下也可复位 |

### 3.5 数码管动态扫描显示

```
┌─────────────────────────────────┐
│   共阳极数码管 (SR410361N)        │
│   4位动态扫描                    │
│                                 │
│   ┌────┐  ┌────┐  ┌────┐  ┌────┐
│   │MIN_H│  │MIN_L│  │SEC_H│  │SEC_L│
│   │ 分十 │  │ 分个 │  │ 秒十 │  │ 秒个 │
│   └────┘  └────┘  └────┘  └────┘
│                                 │
│   位选：三极管 S8550 驱动          │
│   段选：FPGA 直接驱动（限流470Ω）   │
│   扫描频率：≥ 200Hz              │
└─────────────────────────────────┘
```

- 扫描频率：1kHz（每位数码管亮约 250μs）
- 位选信号：经三极管 S8550 放大驱动公共阳极
- 段选信号：a~g + dp，共 8 段，经 470Ω 限流电阻到 FPGA I/O
- 共阳极 → 位选高电平点亮，段选低电平点亮

**7段译码表（共阳极，低电平有效）：**

| BCD | 显示 | a | b | c | d | e | f | g |
|-----|------|---|---|---|---|---|---|---|
| 0000 | 0 | 0 | 0 | 0 | 0 | 0 | 0 | 1 |
| 0001 | 1 | 1 | 0 | 0 | 1 | 1 | 1 | 1 |
| 0010 | 2 | 0 | 0 | 1 | 0 | 0 | 1 | 0 |
| 0011 | 3 | 0 | 0 | 0 | 0 | 1 | 1 | 0 |
| 0100 | 4 | 1 | 0 | 0 | 1 | 1 | 0 | 0 |
| 0101 | 5 | 0 | 1 | 0 | 0 | 1 | 0 | 0 |
| 0110 | 6 | 0 | 1 | 0 | 0 | 0 | 0 | 0 |
| 0111 | 7 | 0 | 0 | 0 | 1 | 1 | 1 | 1 |
| 1000 | 8 | 0 | 0 | 0 | 0 | 0 | 0 | 0 |
| 1001 | 9 | 0 | 0 | 0 | 0 | 1 | 0 | 0 |

> 0 = 亮（低电平），1 = 灭（高电平）

| 信号名 | FPGA引脚 | 方向 | 连接目标 | 说明 |

| clk | 专用时钟脚 | input | 50MHz 晶振 | 系统时钟 |

## 五、设计流程步骤

### 第一步：需求分析与方案确定
- 明确功能需求（5大功能）
- 确定系统架构（模块划分）
- 选定器件型号（参考元件清单）

### 第二步：Verilog 代码设计
- 编写各子模块 RTL 代码
- 编写 Testbench 仿真验证
- 在 Gowin EDA（云源软件）中综合、布局布线

### 第三步：底板原理图设计
- 使用 Altium Designer 绘制原理图
- 引用提供的 `.SchLib` 和 `.PcbLib` 库文件
- 设计：FPGA 核心板的底座接口、数码管驱动电路、按键/开关电路、LED 指示电路、电源电路

- 双面板设计
- 尺寸 ≤ 8cm × 8cm
- 注意：电源走线加粗、去耦电容放置、数字/模拟分区

### 第五步：制板与焊接
- 发厂制板
- 手工焊接元器件

- 逐项测试 5 大功能
- 准备验收答辩

## 六、关键设计要点

1. **时序约束**：50MHz 晶振频率较高，分频器计数器和状态机需要合理设计时序，避免亚稳态
2. **消抖处理**：机械开关/按键的抖动（约 5~20ms）必须在 FPGA 内部做消抖，否则会产生误触发
3. **动态扫描**：扫描频率不能太低（>60Hz），否则人眼会感觉到闪烁
4. **驱动能力**：FPGA 的 I/O 驱动能力有限，数码管位选必须经三极管放大
5. **电源设计**：FPGA 核心板通常需要 3.3V 和 1.2V，底板需提供稳定的 5V 供电，并通过核心板上的 LDO 转换
6. **预置值实现**：可通过拨码开关+按键配合，或通过重新下载程序（修改代码中的初始值参数）

## 七、扩展思路（可选）

- **闹钟功能**：设置目标时间，到达时蜂鸣器提示
- **倒计时模式**：从预置值倒计时到 00:00
- **毫秒显示**：增加两位毫秒数码管，提高精度
- **串口通信**：通过 UART 与 PC 上位机通信，远程控制秒表
- **VGA/LCD显示**：用更大的屏幕显示时间
