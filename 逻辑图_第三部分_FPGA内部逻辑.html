三、模块1：clk_div — 时钟分频器

一个 50MHz 晶振，生出 1Hz 和 1KHz

1Hz → counter_60，每秒触发一次加1。
1KHz → key_debounce 每毫秒采样 + → seg_decoder 驱动动态扫描。

四、模块2：key_debounce — 按键消抖

连续 20 次同值才确认，消除 5~20ms 机械抖动

五、模块3：fsm_ctrl — 控制状态机

IDLE / COUNT / PAUSE 三态，sw_rst 优先级最高

上电默认 IDLE → 显示 00:00, 两个LED都灭 → 等待用户操作。
开关1 优先级最高：无论 COUNT 还是 PAUSE，拨到"清零"立刻归零。

六、模块4：counter_60 — 60进制BCD计数器

4 级联 BCD 计数器：秒个→秒十→分个→分十

级联进位：低位移到最大值 → 下一拍归零的同时产生 carry → 高位收到 carry 后 +1。
预置值：改 Verilog 参数 INIT_MIN 和 INIT_SEC，重新下载。不需要底板硬件改动。

七、模块5：seg_decoder — 显示译码器

BCD→7段译码 + 4位动态扫描

为什么先关后开？切换下一位前先关当前位 → 换段选图案 → 再开下一位。如果先开后换 → 旧图案在新位上闪一下 → 残影。
