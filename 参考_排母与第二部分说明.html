排母是什么 & 第二部分要体现什么

一、排母是什么

核心板下面有一排排的金属针（排针，公头）。底板上面焊了一排排的金属孔（排母，母座）。把核心板往底板上插，针插进孔里，电就通了。

核心板 90x60mm

排针（公头，在核心板底面）
像一排金属腿

→
底板 80x80mm
底板上还有：
◆ 数码管   |   ◆ LED   |   ◆ 开关   |   ◆ 电阻   |   ◆ 三极管
这些器件都通过铜线连到排母的引脚上

排母（母座，焊在底板上）
像一排小孔，等针插进来

排母的作用：核心板上的 FPGA 信号通过 排针→排母 传到底板上的器件（LED、数码管、开关）。
同样，开关的状态也通过 排母→排针 传回给 FPGA。

二、第二部分到底要体现什么

第二部分 = 一张信号接线清单 + 每根线经过什么器件。

它回答的问题是："画 AD 原理图的时候，这根线从 FPGA 出来，串多大电阻，接到哪个器件的哪个脚？"

这就是第二部分：

信号名从哪来到哪去中间经过
SW_RST 拨动开关1 → FPGA 输入引脚 开关闭合→GND；断开→10KΩ上拉到3.3V
SEG[7:0] FPGA 输出引脚 ×8 → 数码管 8 个段选脚 每路串 470Ω
DIG[3:0] FPGA 输出引脚 ×4 → 数码管 4 个位选脚 每路串 4.7KΩ → S8550 基极 → S8550_C接5V → S8550_E接数码管阳极
LED_RUN FPGA 输出引脚 → 绿色LED → GND 串 470Ω
LED_PAUSE FPGA 输出引脚 → 红色LED → GND 串 470Ω
5V 核心板排针 → S8550 集电极 (4路) 走线直接连
3.3V 核心板排针 → 开关上拉电阻 (2路) 走线直接连
GND 核心板排针 → 所有需接地的器件 走线直接连，覆铜铺地

▲ 第二部分就是一个这样的东西。信号名 + 从哪来 + 到哪去 + 经过什么（电阻值/三极管）。

三、第一部分 vs 第二部分 — 本质区别

✓ 第一部分（系统级）

画：一根 Type-C 线 → 整个板子 → 数码管
画：开关1（清零/启动）
画：绿灯→计数，红灯→暂停

不画电阻值、信号名、三极管

给老师/用户看的：
"这个秒表怎么用"

✓ 第二部分（电路级）

画：SEG[7:0] → 470Ω ×8 → 数码管a~g
画：DIG[0] → 4.7KΩ → S8550 → 位1
画：SW_RST → 10KΩ上拉 → FPGA输入

不画器件位置、FPGA内部模块

给画 AD 原理图的人看的：
"这根线怎么连、串多大电阻"

四、一个形象的比喻

第一部分 = 装修方案：
 "客厅放沙发，餐厅放桌子，卧室放床"

第二部分 = 施工图：
 "沙发背后的墙上，离地1.2米处，打一个膨胀螺栓，用M6螺丝固定"

第三部分 = 水管电路图：
 "热水器的冷水管从总阀门分出来，经过前置过滤器，进入热水器进水口"

✗ 第二部分容易犯的错：写太多

"数码管放在底板左上角"
"FPGA IO32 接到数码管 a段"
"走线要 0.5mm 宽"
"代码里 if sw_rst==0 then..."

← 这些分别是 PCB、引脚分配、Verilog 的事
第二部分不要写这些

✗ 第二部分容易犯的错：写太少

"数码管接 FPGA"
"开关接 FPGA"
"LED 接 FPGA"

← 没写电阻值、没写信号名、没写方向
这样画 AD 的人不知道怎么做
