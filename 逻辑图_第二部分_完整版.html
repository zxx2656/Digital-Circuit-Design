二、功能块A：开关电路 — SW_RST, SW_PAUSE

拨动开关 ×2 ＋ 10KΩ 上拉电阻 ×2

×2 路，另一路为 SW_PAUSE

为什么需要上拉电阻？
开关断开时导线悬空，电平随机跳动。10KΩ 接 3.3V，断开时稳在 3.3V。闭合时 GND 拽到 0V，电阻仅流过 3.3÷10000=0.33mA。
为什么是 10KΩ？
太小费电，太大抗干扰差。10KΩ 是标准值。

三、功能块B：LED 指示电路 — LED_RUN, LED_PAUSE

绿色 LED ×1 ＋ 红色 LED ×1 ＋ 470Ω 电阻 ×2

两路电路相同，仅 LED 颜色不同

为什么串 470Ω？
LED 直接接 3.3V 会烧。压降≈2V → 电阻承压 1.3V → 2.8mA。安全（FPGA I/O 上限 8~16mA），亮度适中。
为什么不需要三极管？
一颗 LED 仅 2.8mA，FPGA I/O 直接驱动。数码管位选需约 18mA → 才需要三极管（见功能块D）。

四、功能块C：数码管段选电路 — SEG[7:0]

470Ω 电阻 ×8

为什么每段独立串电阻？
4 位的同名段共享同一条线。只在阳极串电阻 → "1"（2段亮）电流大、"8"（7段亮）电流小 → 亮度不均。每段各串 470Ω：电流恒定，亮度均匀。
3.3V 供电的利弊
好处：FPGA 输出 3.3V 时阳极也是 3.3V → Vbe=0 → PNP 完全关断，零残影。
代价：段电流 2.6mA，比 5V 时暗。4位动态扫描平均 0.65mA/段，室内可辨识。
如需更亮 → 可减小段选电阻（不推荐，物料无更小值）。

五、功能块D：数码管位选电路 — DIG[3:0]

S8550 PNP 三极管 ×4 ＋ 4.7KΩ 电阻 ×4

为什么必须用三极管？
一位全亮（"8"）：7段×2.6mA≈18mA，仍在 FPGA I/O 上限边缘。用三极管更安全：FPGA 只供 0.55mA 基极电流，三极管放大 ≥100 倍 → 集电极可出 ≥55mA。
为什么用 PNP？
共阳极数码管——阳极从 3.3V 取电。PNP 做"高侧开关"：E 接 3.3V，C 接负载。DIG=0V → 导通；DIG=3.3V → 关断。
3.3V 供电的好处：PNP 完全关断
DIG=3.3V 时，Vbe=3.3−3.3=0V，小于 PNP 导通阈值 0.7V → 完全关断，无残影。这是用 3.3V 供电相比 5V 的最大优势。

六、功能块E：底板供电 — 3.3V, GND

核心板排母仅提供 3.3V 和 GND（I/O 电平统一 3.3V）

为什么底板不自带电源？
核心板已有完整供电链路。底板从排母取电，不增加电源芯片。
全板 3.3V 的影响
好处：电路更简单（无需区分 5V/3.3V 网络），PNP 关断可靠。
代价：段选电流 2.6mA（vs 5V 的 5.7mA），数码管偏暗但室内可用。
FPGA I/O 电平匹配：开关、LED、段选、位选全部 3.3V，无需电平转换。
